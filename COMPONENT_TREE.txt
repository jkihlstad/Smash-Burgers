REUSABLE COMPONENTS

├── Used on: All pages
├── Features: Sticky nav, mobile menu, glass background
└── State: Mobile menu open/closed

Footer Component (components/layout/Footer.tsx)
├── Used on: All pages
├── Features: Links, social icons, responsive
└── State: None (stateless)

GlassCard Component (components/ui/GlassCard.tsx)
├── Used on: Menu items, feature cards
├── Features: Liquid glass effects, animations
└── Props: className, children

Button Component (components/ui/Button.tsx)
├── Used on: CTAs, forms, actions
├── Features: Variants, sizes, animations
└── Props: variant, size, className, children

================================================================================
DATA FLOW
================================================================================

App Router (Next.js 15)
    ↓
app/layout.tsx (Root Layout)
    ├── Fonts loaded (Anton, Inter, Jakarta)
    ├── Global styles applied
    ├── Noise overlay rendered
    └── Children rendered
        ↓
    app/page.tsx (Server Component)
        ├── Static data (locations, hours, menu items)
        ├── Header component
        ├── Static sections
        └── Footer component

Client Components (use client):
    - Header (interactive menu)
    - GlassCard (Framer Motion)
    - Button (Framer Motion)

Server Components:
    - app/page.tsx (main page)
    - Footer (static links)

================================================================================
STYLING ARCHITECTURE
================================================================================

Tailwind Config (tailwind.config.ts)
    ├── Liquid Glass colors defined
    ├── Font families with CSS variables
    └── Custom utilities
        ↓
Global Styles (app/globals.css)
    ├── Tailwind base
    ├── Tailwind components
    └── Tailwind utilities
        ↓
Component Styles (inline with cn())
    ├── Responsive classes (sm:, md:, lg:)
    ├── State classes (hover:, active:, group-hover:)
    └── Custom glass effects

================================================================================
IMAGE OPTIMIZATION FLOW
================================================================================

Source: /images/*.JPG
    ↓
Symlink: public/images → /images
    ↓
Next.js Image Component
    ├── Automatic optimization
    ├── Responsive sizing
    ├── Format conversion (AVIF, WebP)
    ├── Lazy loading (except priority)
    └── Blur placeholder
        ↓
Optimized image served to browser

================================================================================
ANIMATION FLOW
================================================================================

Framer Motion Components
    ├── GlassCard
    │   ├── whileHover={{ y: -5, scale: 1.01 }}
    │   └── Glow transitions (opacity)
    ├── Button
    │   ├── whileHover={{ scale: 1.05 }}
    │   ├── whileTap={{ scale: 0.95 }}
    │   └── Shine animation
    └── Image containers
        └── CSS transition-transform (zoom)
